NoLet Sender 隐私权政策

本隐私权政策说明了 NoLet Sender 浏览器扩展如何收集、使用和保护您的信息。我们高度重视您的隐私，并致力于保护您的个人数据。

1. 数据收集和使用

  • 所有数据仅通过用户手动配置 NoLet 服务器的 API URL 发送并且不会与除该 URL 所属域名以外的任何服务器通信
  • 所有数据均存储在用户本地浏览器中
  • 不会收集任何其他个人信息

2. 数据存储

  • 所有配置数据使用浏览器本地存储
  • 不会上传到任何第三方服务器
  • 用户可随时删除所有存储的数据

3. 数据传输

  • 仅在用户触发推送时，将消息内容发送到用户配置的 NoLet 服务器
  • 不会与其他任何服务器通信

4. 用户权利

  • 用户可随时查看、修改或删除已保存的设备配置
  • 卸载扩展会自动清除所有存储的数据

5. 权限使用说明

  • storage: 用于存储用户的设备配置和应用设置
  • contextMenus: 实现右键菜单快速发送功能
  • activeTab: 获取当前标签页中选中的文本
  • clipboardRead: 实现发送剪贴板内容功能
  • notifications: 显示推送结果通知
